# 数据结构

## 常用的数据结构
  - 数组 Array
  - 堆栈 Stack
  - 队列 Queue
  - 链表 Linked Lists
  - 树 Trees
  - 图 Graphs
  - 字典树 Trie
  - 散列表（哈希表） Hash Tables

## 在较高的层次上，基本上有三种类型的数据结构：
  - 堆栈和队列是类似于数组的结构，仅在项目的插入和删除方式上有所不同
  - 链表、树和图结构的节点是引用到其他节点
  - 散列表依赖于散列函数来保存和定位数据

## 在复杂性方面：
  - 堆栈和队列是最简单的，并且可以从中构建链表
  - 树和图是最复杂的，因为它们扩展了链表的概念
  - 散列表和字典树，需要利用这些数据结构来可靠的执行

## 就效率而言：
  - 链表是记录和存储数据的最佳选择
  - 而哈希表和字典树，在搜索和检索方面效果最佳
